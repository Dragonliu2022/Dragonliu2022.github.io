Python字典与字符串互转

2022-02-28 开发 python 0 评论 字数统计: 198(字) 阅读时长: 1(分)

1 字符串转为字典

如果要将字符串转为字典，则可以使用 eval，但是原来双引号会变为单引号。

2 字典转字符串

将字典转为字符串的方式，可以使用 str，但是原来双引号会变为单引号。

使用 json.dumps，就可以保住双引号；如果不用考虑双引号，则使用 str，毕竟比 json.dumps 快了一个数量级。
